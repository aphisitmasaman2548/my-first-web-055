## สิ่งที่ฉันสนใจ

## ความคาดหวังจากการเรียนวิชานี้
1. เขียนเอกสารซอฟแวร์ระบบอย่างเป็นระเบียบเข้าใจง่าย ส่งต่อได้
2. เขียนเอกสารที่สามารถนำไปต่อยอด ทำให้สร้างเว็บไซต์ใหม่ได้ไวขึ้น เรียกว่า Tamplate Docs
3. เข้าใจการเขียนเว็บลึกซึ่งมากยิ่งขึ้นจากการลงมือทำ workshop
